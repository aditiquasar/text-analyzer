# Text Analyzer & Palindrome Checker

A simple Python program that performs basic string operations and checks whether the entered text is a palindrome.


## Features

* Display the first 5 characters of the input text
* Display the last 5 characters of the input text
* Reverse the text
* Count the number of words
* Count the total number of characters
* Check whether the text is a palindrome


## How It Works

The program:

1. Accepts text input from the user.
2. Uses string slicing to extract characters and reverse the text.
3. Uses `split()` and `len()` to calculate word count.
4. Uses `len()` to determine character count.
5. Compares the original text with its reversed version to determine if it is a palindrome.


## Sample Output

Enter text: madam

First 5 chars: madam
Last 5 chars: madam
Reversed: madam
Word count: 1
Character count: 5
Palindrome
